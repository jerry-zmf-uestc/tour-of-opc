# Collaboration

Use this file for runtime coordination and final handoff shape.

## Delegation envelope
Every delegation includes:
- objective
- context or inputs
- deliverable format
- deadline or SLA
- escalation contact

## Runtime protocol
- communication statuses: `accepted / blocked / done`
- task lifecycle: `Inbox / Assigned / In Progress / Review / Done / Failed`
- return-path is mandatory
- non-trivial specialist deliverables must go through Review
- producer cannot approve their own work

## Handoff
Every handoff includes:
- summary
- artifact path
- verification command or acceptance check
- known issues
- next action

## Chat discipline
- no raw bulk output in chat
- chat contains summary plus artifact path only
- deliverables live in `shared/`

## Final report
Include:
- confirmed roles
- workspace layout
- contracts
- workflow summary
- stage deliverables and paths
- validation result
- security summary
- remaining manual actions
- channel binding blueprint
- smoke test prompt
